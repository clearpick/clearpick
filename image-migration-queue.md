## How to use this file

For each product below, go to the Amazon.ca product page, find the main product image, copy the image-id from the URL (the alphanumeric string in `/images/I/{image-id}._AC`), then run:

```bash

# Example: node scripts/migrate-image.js sony-wh-1000xm5 61cR8j0NXML
```

The script updates both `products.json` and the product HTML file in one command. Work through one category at a time.
